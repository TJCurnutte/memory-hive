## How It Works

1. Agents write learnings to `learnings/raw/[agent-id]/`
2. After task completion, agent submits a contribution summary here
3. Curator reviews, promotes to `learnings/distilled/`, or archives
4. All decisions logged to `DECISIONS.md`

## Format for Agent Contributions

```

- ...

- ...
**Mistakes made:**
- ...
**Worth promoting:** Yes / No / Uncertain

---
```
