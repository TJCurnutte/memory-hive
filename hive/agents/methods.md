# Research Methods Guide
*Research-Analyst Agent — Working Reference*

---

## 1. Source Discovery

### Primary Channels
- **Academic databases**: Google Scholar, PubMed, JSTOR, arXiv, SSRN, IEEE Xplore
- **Government & official**: data.gov, census.gov, agency publications (.gov / .govt)
- **Think tanks & NGOs**: Brookings, RAND, Pew, McKinsey, World Bank, IMF, UN reports
- **Industry reports**: Gartner, Forrester, Statista, IBISWorld (often paywalled — note access limits)
- **News**: Reuters, AP, BBC, The Atlantic (established, editorial standards — not tabloids)
- **Expert networks**: Clarity, Guidepoint, AlphaSights (for primary expert consultation)
- **Community intelligence**: HN, Reddit (r/AskScience, r/credibledefense), specialized Slack/Discord

### Search Strategies
- Start broad, narrow with specificity (e.g., "AI regulation EU" → "EU AI Act compliance requirements 2024")
- Use filetype:pdf for reports, site:gov for government sources
- Track citations backward (snowball) and forward (who cited this paper?)
- Set up Google Alerts for ongoing topics

## 2. Source Evaluation — CRAAP+

| Criterion | What to Check |
|---|---|
| **Currency** | Publication date, when data was collected, last updated |
| **Relevance** | Does it directly address your question? Same context/region/sector? |
| **Authority** | Who authored it? Credentials, institutional affiliation, prior work |
| **Accuracy** | Are claims supported by citations? Peer-reviewed? Methodology stated? |
| **Purpose/Point of View** | Advocacy? Sponsored content? Political framing? Bias acknowledged? |
| **+ Coverage** | Does it cover the full scope, or cherry-pick aspects? |

### Peer Review Signals
- Journal indexed in PubMed, Scopus, Web of Science
- DOI and ORCID for authors
- Clear methodology section
- Acknowledged limitations
- Funding sources disclosed

## 3. Cross-Referencing & Triangulation

- **3-source minimum** for factual claims: find 3 independent sources making the same claim
- **Diversify source types**: academic + government + news (not just 3 news articles)
- **Check consensus vs. debate**: if sources disagree, surface the disagreement explicitly
- **Forward citations**: use Google Scholar "cited by" to find newer corroborating evidence
- **Negative search**: search for the counterargument to ensure you haven't lopsided your findings
- **Raw data vs. synthesis**: primary sources (raw data, original studies) > secondary (analysis of primary)

## 4. Red Flags

### Misinformation Patterns
- Single-source claims with no corroboration
- Emotional language replacing evidence ("shocking," "they don't want you to know")
- Missing dates, missing author, missing methodology
- Conspiracy frameworks ("everyone is covering this up")
- Vague attributions ("experts say," "studies show" with no citations)

### Predatory Journals
- Sound-alike names to legitimate journals (e.g., "Journal of Advanced Research" vs predatory)
- No clear editorial board listed
- Very fast publication timelines
- Unsolicited email invitations to publish
- Check: Beall's List, DOAJ (whitelist), CrossRef

### Sponsored / Conflict-of-Interest Content
- "Sponsored content" or "partner content" labels in news
- Industry-funded studies (tobacco, pharma, tech lobbying reports)
- Think tank funding sources listed in small print

## 5. Credibility Hierarchy

```
1. Peer-reviewed primary research (randomized controlled trials, meta-analyses)
2. Government statistical agencies (census, BLS, CDC, Eurostat)
3. Reputable academic institutions (Harvard, MIT, Stanford research)
4. Established NGOs / think tanks with methodology transparency
5. Professional journalism (Reuters, AP, The Guardian, NYT, WSJ)
6. Trade publications / industry analysts (Gartner, Forrester)
7. Company reports / press releases (treat as primary source, verify externally)
8. Blogs / personal websites (only if expert author with verifiable credentials)
9. Social media / forums (use for leads, never as primary source)
```

*Lower on the list is not useless — it's a starting point for chasing higher-quality sources.*

## 6. Practical Workflows

### Quick Research (< 30 min)
1. Define the exact question in one sentence
2. 3 Google searches: broad → specific → counterargument
3. Open 5 tabs max; discard if not immediately relevant
4. Check Wikipedia for overview + citations (verify each citation)
5. Write 3-sentence answer with source links

### Deep Research (2+ hours)
1. Clarify scope, audience, and output format before starting
2. Build a source matrix: source, type, credibility rating, key finding, URL
3. Read 10–15 sources, tag by theme
4. Identify: consensus points, debate points, gaps, outliers
5. Draft structure → fill in → cite inline → review for balance
6. Peer sanity-check: have someone poke holes before delivery
