# PART 1: 18 CONTENT SKILLS

## Content Skill 1: The Code Demo Post
**Format:** Screenshot/GIF + short caption
**Hook:** "Watch this agent do X in 30 seconds"
**Structure:** Tweet 1 (hook) + Tweet 2 (screenshot) + Tweet 3 (how to build) + Tweet 4 (CTA)
**Best for:** Developer audience, tool demos, workflow showcases
**Timing:** Tue–Thu, 8–10am EST

## Content Skill 2: The "Nobody Talks About X" Thread
**Format:** 5–8 tweet thread
**Hook:** "Nobody talks about X. They should."
**Structure:** Tweet 1 (contrarian claim) + Tweets 2–6 (supporting points) + Tweet 7 (why now) + Tweet 8 (CTA)
**Best for:** Thought leadership, establishing authority

## Content Skill 3: The Listicle Thread
**Format:** Numbered list, each tweet is one item
**Hook:** "X tools I use daily" / "X ways to do Y"
**Structure:** Intro tweet + Tweets 2–6 (each item) + Tweet 7 (summary) + Tweet 8 (CTA)
**Best for:** Value delivery, resource sharing, evergreen content

## Content Skill 4: The ROI Metrics Post
**Format:** Single image/text post with bold claim
**Hook:** "My AI agents handled X% of my outreach last month"
**Structure:** The metric (bold) + context + implied process + invitation
**Best for:** Proving value, attracting operators and founders
**Timing:** Wed or Thu, mid-day

## Content Skill 5: The Build-in-Public Log
**Format:** Recurring series (weekly or bi-weekly)
**Hook:** "Ship log: week X" / "What I shipped this week"
**Structure:** Tweet 1 (week + theme) + Tweets 2–4 (key updates) + Tweet 5 (lessons) + Tweet 6 (next week)
**Cadence:** Same day each week to build expectation

## Content Skill 6: The Hot Take
**Format:** Single strong tweet or 2-tweet thread
**Hook:** "Hot take:" / "Unpopular opinion:" / "Controversial:"
**Structure:** Tweet 1 (hot take) + Tweet 2 (nuance or defense, optional)
**Best for:** High engagement, debate, algorithmic boost
**Risk:** Stay in professional lane

## Content Skill 7: The Comparison Thread
**Format:** Side-by-side analysis, 8–12 tweets
**Hook:** "X vs Y — which one should you actually use?"
**Structure:** Setup + Tweets 2–8 (breakdown) + Recommendation + Engagement prompt
**Best for:** SEO value, decision-stage audience

## Content Skill 8: The Tool Announcement Post
**Format:** Single post or 2-tweet thread
**Hook:** "PSA:" / "Just shipped:" / "New in OpenClaw:"
**Structure:** What was built + Why it matters + How to access + Link/CTA
**Timing:** Tue or Wed morning

## Content Skill 9: The Before/After Post
**Format:** Two-panel image or two-part text
**Hook:** "My workflow before OpenClaw vs after"
**Structure:** Tweet 1 (before — messy, slow) + Tweet 2 (after — clean, automated) + Tweet 3 (takeaway)
**Best for:** Conversion-focused content, tool evaluation stage

## Content Skill 10: The Quote/Insight Drop
**Format:** Single quote tweet or static image
**Hook:** "This is the future of X" / screenshot of a great line
**Structure:** The insight (1–3 sentences) + Attribution + Minimal context
**Best for:** Thought leadership, intellectual engagement

## Content Skill 11: The AMA / Open Thread
**Format:** "Ask me anything about X" + thread
**Hook:** "Ask me anything about running outreach with AI agents"
**Structure:** Tweet 1 (the ask) + Replies (answer each) + Tweet N (summary)
**Best for:** Community building, authority, low-effort high-engagement

## Content Skill 12: The Meme / Relatable Post
**Format:** Single image or GIF meme
**Hook:** "POV: trying to use X without OpenClaw" / relatable dev humor
**Structure:** Humorous image/GIF + Optional 1-line caption
**Frequency:** 1–2x per week max, keep professional

## Content Skill 13: The Customer Story / Use Case
**Format:** 6–10 tweet narrative thread
**Hook:** "This founder used OpenClaw to do X in Y time"
**Structure:** Problem + Search for solution + How OpenClaw was used + Result (quantified) + Lesson
**Best for:** Social proof, conversion, case study content

## Content Skill 14: The Resource / Toolkit Post
**Format:** Single post or 3-tweet thread
**Hook:** "The complete X toolkit for Y in 2026"
**Structure:** Intro + Tweet 2–5 (numbered tools/resources) + "Drop your favorites below"
**Best for:** Evergreen, high-share, backlink value

## Content Skill 15: The Personal Story / Vulnerability Post
**Format:** Single post or short thread
**Hook:** "I almost quit because of X" / "The mistake I made cost me Y"
**Structure:** The mistake + What you learned + What you would do differently + How you moved forward
**Best for:** Authenticity, community trust, differentiation

## Content Skill 16: The Poll / Engagement Prompt
**Format:** Single poll or question tweet
**Hook:** "Which would you rather have:" / "Has this happened to you?"
**Structure:** The question + Follow-up results post (24–48h later) + Insights from results
**Best for:** Algorithm boost, community insight gathering
**Frequency:** 1x per week

## Content Skill 17: The Anti-Pattern Thread
**Format:** 7–10 tweet thread
**Hook:** "Stop doing X" / "The X mistake that costs you Y"
**Structure:** Warning + Examples of the mistake + Corrections + Right approach + Engagement prompt
**Best for:** Problem-aware audience, high engagement

## Content Skill 18: The Weekend Read Long-Form Post
**Format:** Single text post or thread
**Hook:** "A thread on X that will save you hours" / "Everything you need to know about X"
**Structure:** Comprehensive overview + Deep examples + Actionable takeaways
**Best for:** Evergreen resource, high-save rate, thought leadership
**Timing:** Saturday or Sunday evening

# PART 4: TIMING TACTICS

## Optimal Posting Windows

| Platform | Best Days | Best Times (EST) | Avoid |
|----------|---------|-----------------|-------|
| X/Twitter | Tue–Thu | 8–10am, 12–2pm, 6–8pm | Mon before noon |
| LinkedIn | Tue–Thu | 7–9am, 12–1pm | Mon all day, Fri afternoon |
| DEV.to | Mon, Wed, Fri | 6–8am | Weekends |
| Bluesky | Daily | 9–11am, 7–9pm | None yet |

## Algorithm Optimization:
- Post engagement-heavy content in morning to maximize afternoon replies
- Post evergreen "resource" content Sunday evening for Monday shares
- Always reply to early comments within 30 minutes
- Pin the highest-performing tweet of any thread

---

# PART 5: ENGAGEMENT OPTIMIZATION

## The 30-Minute Rule:
After posting, spend 30 minutes engaging in replies to boost algorithm signal.

## Reply Strategy:
- Reply with added value, not just "great point!"
- Turn replies into mini-threads if the question is good
- Ask a follow-up question in every 3rd reply

## Quote Tweet Strategy:
- Always add 2+ lines of original commentary when QT-ing
- Never just react — always interpret or build on the original
