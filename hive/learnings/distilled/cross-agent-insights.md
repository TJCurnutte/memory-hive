# Cross-Agent Insights

Insights that emerged from the interaction of multiple agents or domains. Things that wouldn't be visible from any single agent's perspective.

---

## How to Add Entries

When a cross-agent insight is identified, add it here with:
- What the insight is
- Which agents or domains contributed
- How it connects different parts of the system
- Date first identified
- Implications for future work
